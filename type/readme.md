Generell: *.bas are basic programs. So basic (E000R) must be running before "typed"/loading.
*.txt are machine programs in woz hex format to be typed/loaded when woz mon is running.

The machine programs must be started with ADRR R after loaded.
```
mandel3.bas		
reverse.bas
fib.bas	
dobble4k.bas	

APPLE30TH.txt	- 280
hello_world.txt	- 280
mastermind.apl.txt - 300
mandelbrot-65.txt - 280

To load such a program:
CTRL+L
Filename: type/hello_world.txt
1 (ASCII)
1 (Type)
```
